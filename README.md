# 🎮 Race Stars

## 📌 Overview
Race Stars is a vibrant low-poly arcade racing game where style meets speed. Burn rubber on twisty tracks, drift past your rivals, and aim for the podium in a race where every second counts. With simple visuals and smooth controls, Race Stars brings back the joy of classic racing — no licenses, no rules, just clean competitive fun.
Whether you're aiming for the fastest lap or just trying not to spin out, this one's for every fan of fast wheels and low-poly thrills.

---

## 🧪 How to Play / Run the Game
1. Clone or download this repo
2. Open it in Unity
3. Press `Play` to start in the Unity Editor

> Optional: You can also download and play the standalone build from https://piyushch13.itch.io/racestars
